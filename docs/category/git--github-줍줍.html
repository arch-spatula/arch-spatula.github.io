📄️ git stash

git pull을 하려고 하는데 그 전에 조금 여기저기 끄적거렸습니다. 그냥 pull하면 문제된다면 이럴 때 그냥 git stash하고 pull하면 문제 없습니다. 물론 커밋을 해도 되지만 커밋하기 위해 생각하는 노력이 귀찮으면 이렇게 하면 됩니다.
