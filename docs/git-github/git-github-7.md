# 정리한 비어있는 폴더 공유하기 `.gitkeep`

```
/(폴더이름)
  .gitkeep
```

폴더 속에 `.gitkeep` 파일을 만들고 커밋하면 폴더 구조다 레포에 올라갑니다.

예전에 비어있던 `index.ts`하고 올리던 짓거리를 이제 안 하고 사람 비슷한 흉내를 수 있습니다.
