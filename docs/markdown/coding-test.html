2. 풀이 전 계획과 생각​

  • N을 int로 바꾸는 것 까먹지말기
n = int(input())
  • print문을 9번 쓰기
print(n * 1)
print(n * 2)
print(n * 3)
...

3. 풀이​

n = int(input())
print(n*1)
print(n*2)
print(n*3)
print(n*4)
print(n*5)
print(n*6)
print(n*7)
print(n*8)
print(n*9)
