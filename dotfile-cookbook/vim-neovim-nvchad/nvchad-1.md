# nvchad

nvchad에서 사용할 수 있는 이런 저런 커맨드입니다.

## 검색

- `leader` + `ff`/`fh`/`fb`/`fw`: 파일/헬프/버퍼/단어 검색
- `leader` + `th`: 테마 검색

## 터미널 접근

- `leader` + `w`: 세로 터미널 열기
- `leader` + `v`: 가로 터미널 열기
- `exit` 커맨트로 탈출할 수 있습니다.

## 기타 명령

- `leader` + `/`: 주석처리
- `ctrl` + `n`: 자동완성 추천
