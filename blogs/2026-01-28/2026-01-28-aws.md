# 쉽게 설명하는 AWS 기초 강의

- [쉽게 설명하는 AWS 기초 강의](https://www.inflearn.com/course/%EC%89%BD%EA%B2%8C-%EC%84%A4%EB%AA%85%ED%95%98%EB%8A%94-aws-%EA%B8%B0%EC%B4%88)를 수강하기 시작했습니다.
- 운동을 하는 날에는 1강의 이상
- 운동을 안 하는 날에는 2강 이상 듣고자 합니다.
- 주말, 휴일에는 3강 이상을 듣고자 합니다.

- 강의를 듣기 시작하기로 한 이유가 있습니다. 소프트웨어 엔지니어링에서 실제 코드 작성의 비중이 많이 줄어든다면 그 외 부분 중 하나로 AWS 지식이 적절할 것 같다는 생각이 들어서 듣기 시작했습니다. 프론트엔드, 백엔드 2분법적인 관점이 아니라 제품을 만들고 기술로 비즈니스 문제를 해결한다는 관점에서 제가 알아야할 기술 지식을 늘리고자 AWS가 적절할 것 같다는 생각이 들었습니다. 범위를 더 얿히고자 했습니다.

- 전문성을 생각해서는 <밑바닥부터 시작하는 웹 브라우저>를 읽고 예제를 따라하는 것이 적절할 것 같습니다. 브라우저의 내부 구현 디테일이 무엇인지 탐구해보는 활동은 저에게 필요합니다.

- 섹션별로 나눠져있습니다.
- 일단 섹션 2의 강의에서 오랫만에 CS 지식 설명이 좋았습니다.

- 바로 MFA 설정부터 다룬 점이 적절하다는 생각이 들었습니다.
- 저는 개발을 처음 시작하면서 AWS 계정을 만들었습니다. 하지만 계정 활용은 거의 안 했습니다.
